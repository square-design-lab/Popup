How to Trigger
Add the prefix to any link's URL field:
#sdl-popup=/about — full page
#sdl-popup=/about#team — a section by ID
#sdl-popup=/about.fe-block-abc — a Fluid Engine block
Use the /#sdl-popup= form if Squarespace strips the leading #.
